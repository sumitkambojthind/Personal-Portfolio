What I've Built

🧩 Fusion 360 Add-In

CAD · AI · Plugin

SurfPick — Smart Surface Selection

A Fusion 360 add-in that works like GitHub Copilot for CAD surface selection. Click one face and the plugin automatically suggests all other surfaces belonging to the same feature (hole, pocket, slot, boss) — press Enter to accept, Escape to reject. Targets CAM programmers working with imported STEP/IGES geometry where feature history is lost.

Python Fusion 360 API BRep Topology Feature Recognition Graph Algorithms

⚙️ CAM Automation · 6 months

ML · CAD/CAM · Automation

AFR — Auto Feature Recognition

A Python-based ML system that automatically extracts and classifies geometric features from CAD models (STEP/AP242 format) to assist CAM automation. Classifies features into Holes, Closed Pockets, Open Inner/External, and Protrusions using a combination of rule-based geometry analysis and an LLM classifier — eliminating the need for manual data entry in manufacturing workflows.
